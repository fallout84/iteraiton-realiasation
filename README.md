Реализация численного решения дифференциальных уравнений первого порядка вида $y' + ky = f$ с использованием центральных разностей и итерационных методов.

## 📌 О проекте

Данный проект демонстрирует полный цикл разработки вычислительного ПО: от математической постановки и генерации СЛАУ до оптимизированного решения на C++ и анализа в Jupyter Notebook.

### Математическая база
* **Дискретизация:** Центральная разность второго порядка $O(h^2)$.
* **СЛАУ:** Формирование трехдиагональной матрицы $A$ и вектора правой части $b$.
* **Метод решения:** Итерационный метод.
* **Оптимизация:** Динамический выход по невязке ($\|r\| < \varepsilon$).
